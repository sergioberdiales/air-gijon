Título: Aplicación Web para la Consulta y Predicción de la Calidad del Aire en Gijón

Descripción corta: El proyecto consiste en desarrollar una aplicación web que permita a cualquier usuario consultar la calidad del aire en Gijón, mostrando tanto la situación actual como la predicción a un día vista. La aplicación se apoyará en datos oficiales proporcionados por el Ayuntamiento (específicamente de la estación ubicada en la Avenida de la Constitución) y se centrará en el contaminante PM10. Se ofrecerán funcionalidades diferenciadas según el tipo de usuario: ciudadanos anónimos podrán consultar la información, mientras que los usuarios registrados podrán suscribirse a notificaciones y alertas, y los gestores internos tendrán la capacidad de actualizar contenidos y realizar un seguimiento de las predicciones mediante informes.


Enumerar el Software y Hardware necesario para su desarrollo, así como las plataformas donde se ejecutará el proyecto. (no superar la cara de esta hoja)


Frontend: HTML5, CSS3 y JavaScript, aplicando un enfoque “Mobile First” para asegurar la compatibilidad con dispositivos móviles.
Backend: Node.js, utilizando el framework Express para la creación de APIs y el manejo de la lógica del servidor, gestión de la base de datos, etc.
Modelos de Predicción: Los algoritmos de aprendizaje automático se desarrollarán en Python y se integrarán con el backend mediante servicios web o módulos específicos.
Base de Datos: MySQL o similar. Para almacenar tanto los datos de contaminación como los registros de usuarios y predicciones.
Despliegue / Puesta en Producción: Servidor de alojamiento (VPS o servicio de hosting) para desplegar la aplicación y realizar pruebas en un entorno real.
